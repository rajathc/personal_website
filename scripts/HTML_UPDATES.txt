NOTES:
================================================================================

1. Video attributes explained:
   - autoplay: Starts playing immediately (like GIF)
   - loop: Repeats infinitely (like GIF)
   - muted: Required for autoplay to work
   - playsinline: Plays inline on iOS (doesn't go fullscreen)

2. The <img> fallback ensures old browsers still show the GIF

3. WebM listed first because it's smaller than MP4

4. Style matches your existing CSS for images (border-radius, margin)

5. After updating and testing, you can delete the original GIF files to save space

6. Jekyll will auto-reload when you save the HTML files
